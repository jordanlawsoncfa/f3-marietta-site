### Definition
A team-building run where the group stays in a tight formation.

### How it's done
The PAX run in a single file line at a steady pace. The man at the back of the line sprints to the front to become the new leader. This repeats until the entire line has rotated through the front.

### Variations
- Indian Run

### Notes
Excellent for 'Picking up the Six.'
