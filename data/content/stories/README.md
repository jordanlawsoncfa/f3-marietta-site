### Purpose

Testimonials, impact stories, and history of the region.

### File format

- One concept or topic per .md file.
- All files should include frontmatter like:

\`\`\`md
---
title: Example Title
category: Stories
tags: [tag1, tag2]
aliases: []
---
\`\`\`

### Examples

- `founding-story.md`
- `pax-testimony.md`
