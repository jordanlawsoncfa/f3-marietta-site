### Purpose

Definitions and explanations of F3 terminology and cultural concepts (the language of F3).

### File format

- One term or concept per .md file.
- All files should include frontmatter like:

```md
---
title: Term Name
slug: term-slug
category: Lexicon
tags: [f3, marietta, lexicon]
---
```

### Examples

- `fng.md`
- `pax.md`
