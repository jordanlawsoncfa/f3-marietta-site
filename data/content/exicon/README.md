### Purpose

The Exercise Lexicon. Definitions and execution guides for exercises performed in F3 workouts.

### File format

- One exercise per .md file.
- All files should include frontmatter like:

```md
---
title: Exercise Name
slug: exercise-slug
category: Exicon
tags: [f3, marietta, exicon]
---
```

### Examples

- `burpee.md`
- `merkin.md`
