### Definition
A 4-count jumping jack.

### How it's done
1. Jump feet out, hands up
2. Back to start
3. Out
4. Back to start

### Variations
- Smurf Jacks
- Seal Jumps

### Notes
Standard warmup move.

- Smurf Jack
